Measures of Central Tendency Equations

In statistics, measures of central tendency help us understand the typical or central value in a dataset. The three main measures are mean, median, and mode. Each provides different insights into your data.

1. Mean

The mean is the average of all numbers in a dataset. It's calculated by summing all values and dividing by the count of values.

Mean = (Sum of all values) / (Number of values)

In mathematical notation, this is often written as:

x̄ = (Σx) / n

Where:

  • x̄ (pronounced "x-bar") is the mean
  • Σ (sigma) means "sum of"
  • x represents each value in the dataset
  • n is the number of values

Example:

Dataset: 2, 4, 4, 4, 5, 5, 7, 9

Mean = (2 + 4 + 4 + 4 + 5 + 5 + 7 + 9) / 8 = 40 / 8 = 5

Note: The mean is sensitive to extreme values (outliers) and may not always represent the "typical" value well in skewed distributions.

2. Median

The median is the middle value when a dataset is ordered from least to greatest. For an odd number of values, it's the center value. For an even number, it's the average of the two central values.

Median = Middle value (odd n) or Average of two middle values (even n)

For an odd number of values:

Median = x[(n + 1) / 2]

For an even number of values:

Median = (x[n/2] + x[(n/2) + 1]) / 2

Where:

  • x[i] represents the ith value in the ordered dataset
  • n is the number of values

Examples:

Odd number of values: 2, 3, 4, 5, 6, 8, 9

Median = 5

Even number of values: 1, 3, 5, 6, 7, 8

Median = (5 + 6) / 2 = 5.5

Note: The median is less affected by outliers and is often used for skewed data or when extreme values are present.

3. Mode

The mode is the value that appears most frequently in a dataset. A dataset can have one mode, more than one mode (multimodal), or no mode.

Mode = Value(s) with highest frequency

There's no specific equation for calculating the mode, but it can be found by:

  1. Counting the frequency of each unique value in the dataset
  2. Identifying the value(s) with the highest frequency

Examples:

One mode: 1, 2, 3, 3, 4, 5 (Mode = 3)

Two modes: 1, 2, 2, 3, 4, 4, 5 (Modes = 2 and 4)

No mode: 1, 2, 3, 4, 5 (each number appears once)

Note: The mode is the only measure of central tendency that can be used with nominal (categorical) data.

4. Weighted Mean

The weighted mean is used when some values in a dataset are more important than others. Each value is multiplied by its weight before summing.

Weighted Mean = (Sum of weighted values) / (Sum of weights)

In mathematical notation:

x̄w = (Σwx) / (Σw)

Where:

  • x̄w is the weighted mean
  • w represents the weight for each value
  • x represents each value in the dataset

Example:

Values: 2, 4, 6 with weights 1, 2, 3 respectively

Weighted Mean = (2*1 + 4*2 + 6*3) / (1 + 2 + 3) = 26 / 6 = 4.33

5. Geometric Mean

The geometric mean is used for data that is multiplicative in nature, like growth rates or returns.

Geometric Mean = (x1 * x2 * ... * xn)^(1/n)

In mathematical notation:

GM = (Πx)^(1/n)

Where:

  • Π (pi) means "product of"
  • x represents each value in the dataset
  • n is the number of values

Example:

Growth rates: 1.05, 1.03, 1.06

Geometric Mean = (1.05 * 1.03 * 1.06)^(1/3) ≈ 1.0465

Choosing the Right Measure:
  • Use the mean for symmetric distributions without significant outliers.
  • Use the median for skewed distributions or when outliers are present.
  • Use the mode for categorical data or to find the most common value.
  • Use the weighted mean when values have different levels of importance.
  • Use the geometric mean for data involving growth rates or percentages.
